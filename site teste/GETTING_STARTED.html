Passo 1: Configurar Variáveis

Configure no Netlify Dashboard ou arquivo .env:

DISCORD_WEBHOOK_URL=seu_webhook_discord
SITE_URL=https://seu-dominio.com

Passo 2: Teste Localmente

Use Netlify Dev para simular produção:
