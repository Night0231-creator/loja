   → Acesse: Netlify Dashboard
   → Settings → Environment → Site variables
   → Adicione:
      MP_ACCESS_TOKEN = [seu token Mercado Pago]
      DISCORD_WEBHOOK_URL = [seu webhook Discord]
      SITE_URL = https://seu-dominio.com

2. TESTAR LOCALMENTE
   $ npm install -g netlify-cli
   $ netlify dev
   → Acesse: http://localhost:8888

3. TESTAR FLUXO COMPLETO
   ✓ Selecione um pacote
   ✓ Clique "FINALIZAR COMPRA"
   ✓ Confirme no modal
   ✓ Deve redirecionar para Mercado Pago
   ✓ Use cartão: 4111 1111 1111 1111 (teste)
   ✓ Após aprovação: Tela de sucesso
   ✓ Discord deve receber notificação

4. FAZER DEPLOY
   $ git add .
   $ git commit -m "refactor: v2.0"
   $ git push origin main
   → Netlify faz auto-deploy (30s)

5. VALIDAR PRODUÇÃO
   ✓ Site abre em: https://seu-dominio.com
   ✓ HTTPS ativo
   ✓ Fluxo funciona
   ✓ Discord recebe notificações
